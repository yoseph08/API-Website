About My API Website

Technology

This site is built using HTML, CSS, and JavaScript. APIs are integrated using the Fetch API for asynchronous data handling.

Objective

My goal was to build a responsive, easy to use interface that showcases how APIs can be integrated smoothly using only front-end technologies.

Advice Page

Integrates Advice Slip API to display random advice.

Fun Facts Page

Uses the Useless Facts API to provide random trivia facts.

Excuse Generator

Connects with the Excuser API to generate random excuses.

Joke Generator

Uses the JokeAPI to fetch various joke formats and categories.
